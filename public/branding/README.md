# Client branding assets

## Your logo (active)

Place your main logo here:

| File | Status |
|------|--------|
| **`logo.png`** | ✅ Your DARBIO CONSULTS LIMITED logo is installed |

The site loads **`/branding/logo.png`** automatically in the **navbar** and **footer**.

## Where it appears

- Top navigation (left side)
- Footer
- White rounded backing keeps the logo visible on dark hero backgrounds

## Optional files

| File | Purpose |
|------|---------|
| `logo-icon.png` | Square icon only (favicon / small spaces) |
| `favicon.ico` | Browser tab — copy to `/public/favicon.ico` or replace `app/icon.svg` |

## Replace the logo

1. Save a new image as **`logo.png`** in this folder (PNG or JPG, transparent or white background both work).
2. Refresh the browser (hard refresh: Cmd+Shift+R).

Recommended size: **width 400–800px**, full logo with icon + text.
